使用日程本用于更好的生活学习工作和复盘

发表于2020-12-01|更新于2020-12-11|生活GTD
|阅读量:

写一篇这类型的博客，我就是在和自己对话，不断地问自己怎么办，该怎么办，怎么做更好。写下来的博客第一位是给自己看，可能我后面的这关有关自我思考博客都会是类似这种，自己这方面目前也还没有到优化传播的阶段，完善自身作为这类型博客的目的。如果有幸遇到了你，如果有辛你看到，希望我的经历能给你有所灵感或启发，如果有什么好的想法和见解可以留言呐，祝你在自己独一无二的路上越走越好。

我为什么选择日程本来记录

其实现在记录生活和待办事项的方式有很多，便利单，手账本，GTD效率手册，各类不同的日程本很多种。网上也有各种软件，如滴答清单，板栗看板…一大堆产品。试用过很多，也付费了不少，但最终这笔钱也都打了水漂😂。

我仔细思考了一下自己的问题和习惯：

  • 首当其冲就是懒；
  • 因为懒，太复杂的终归就是三分钟热度；
  • 针对自身使用的目的，各种软件和记录本有很多冗余的功能或功能缺失；
  • 相比用软件，个人更喜欢用纸笔来记录，有助于自己思考；
  • 个人偏理性，加上手残，手账这类需要制作型的不太适用；

所以尝试了很多都没能坚持使用。但自身觉得确实有用，也有必要，也结合看过的GTD相关书籍，最终选择经济适用的，最普遍的日程本，在此基础上建立自己的使用方式。因为功能不多，拥有富余空间，所以也就有了用简单的方式添加功能的空间。

两本日程本区别开生活与工作

但是日程本我选择购买了两本，一本黑橙色用于生活学习和开支记录，一本黑银色的用于工作，毕竟现在也已经成为了一个“光荣”的打工人。主要是为了将工作与生活明确区分开来。

![two book](E:\hexo\hexo-blog-fly\source_posts\正确使用日程本更好的生活学习和复盘\two book.jpg)

生活橙本（黑橙色生活日程本的简称）在我这里的定位就是老老实实记录好生活、学习、项目、财务、健康和人际关系六个方面的to-do list，以及每天大大小小的收支情况。以周为单位进行计划安排，每日页面用于执行，每月页面用于总览。并进行简单的日小结与周汇总，和相对较为详细的月度总结记录。

工作银本（黑银色工作日程本的简称）就是记录好每天的工作事项，总结工作经验用于学习复盘。同样是以周为单位，每日页面用于执行，每月页面用于总览。但是这个的目的并不是应付工作，而是为了自己成长，不记录无意义的东西。所以页面中记录的更多偏向于工作感悟，方法、经验，和对于后续工作相关方向的思考等。

相辅相成的其他工具

还有一个普通的横条本用作草稿本，用于辅助思考，并且针对各种一次无法完成的事项【项目】进行阶段性计划，保证它能正常落地。

收集箱用于收集待办事项和灵感这些突然想到的事情，我选择了使用【板栗看板】 后面等我有更多的使用经验会单独推荐一下，有兴趣的同学可以网上搜索了解一下。目前还没有正式推广，但使用体验不错。

博客我用来当记笔记的地方，用于思考、记录、分享和复盘。同时可以随用随取。而自己建一个博客功能可以自定义，也可以免受其他平台的打扰。

生活橙本核心部分使用方法

我自己的生活橙本核心部分分为人生规划，月初页，每日页，周总结计划和月度总结五部分。本子里还有如21天习惯养成，项目进度，大额支出甚至加班记录等部分，都有些自己的使用习惯，但就不一一罗列了。

人生规划页

人生规划页分为三年和一生两个区域，仔细思考想一想记下来。有助于明确自己的方向，坚定自己的信念，给与动力。

我是在脑海中自问“我这一生最想要什么”，“做到什么”“还有什么”等问题，然后在草稿本上记录下来，仔细想想那个对自己更重要，排个序。最后摘抄到日程本此页。

月初页

月初页分有事项区，待办列表和月历区三大部分组成：

  • 项目进度：我主要用于放置周期较长的事项。
  • 待办列表：此处只放几件最核心重要的事，因为我有板栗看板作为收集箱，
    • 可以是本周的学习内容；
    • 可以是要做的项目拆解目标；
    • 判定标准参照：此处放的事，哪怕只有三四件事都需要你好好计划安排一下了。
  • 月历区：放各种纪念日、承诺和其他重大事项节点；
    • （重复性事项根据自己实际情况考虑是否需要放置）

每日页

因为橙本只是记录生活学习和收支事项，刨去工作时间。

因为我的上班时间比较诡异，下午1点到8点半。所以我将上部放成了事项区，通常放学习或者项目事项。事项区按照时间进行安排具体事项。

下面还有一个生活事项区，因为生活琐事大多数事情都可以用碎片时间顺道完成，所以主要用于提醒。如果花费时间较长例如大扫除，也可放在上面。

日收支区介于总结与事项区之间，如要是日预算，收入和支出三部分。前部写完总收支情况，在尾部往下具体按吃饭、出行等大分类记录一下自己的钱都花在了哪几个类目下面。不计详细清单，按大分类复盘。

日收支每晚回来睡前复盘记录，第二天可能就忘记了。因为现在多用微信和支付宝，我一般用支付宝，所以直接查看当日账单即可。

日总结区每晚进行简单的复盘，想到什么写什么，比如今天吃饭花的有点多，能不能控制点。今天什么做的很好，要坚持啊，当成小日记来写也可以呢。对于我这种懒人来说，我从一开始就在脑子里告诉自己这不是一个任务，随便记记，想到什么写什么就好，如果太复杂肯定不会长久的，哈哈。

周总结+周计划

周总结+周计划有点尴尬，因为这个本子它默认周六周天放在了一页，但是我周内休息。

所以我决定周末不用处理太复杂的事情，可以睡个懒觉，玩玩游戏看看剧啊，收拾一下房子啊，出去采购一下啊撒的。如果有比较急的事情也可以趁这个时间去忙。

但周总结和周计划要好好记一记：

  • 针对本周的收支情况做一个统计和复盘分析；
  • 根据本周收支情况，结合月预算对下周的日常做一个预算；
  • 针对本周的事情完成度做一个整理；
  • 针对完成度对下周事项做一个安排；
  • 全局上本周有什么好的点，有什么问题。比如
    • 时间安排不合理
    • 感觉安排太多太累了
    • 浪费的时间太多
    • 等等
  • 结合上一点，思考下周计划如何安排，解决问题。
  • 本月目标完成度如何，按照这个进度是否能完成？
  • 除了在日程本上记录，可以针对本周学习的内容，项目等东西做一个整理；
  • 可以从生活、健康、关系看看有没有需要记录的点，就是脑子里一问就立马能冒出来的点。

核心的点一定要记，这几项一定要记。相比日记的感性随意，周记需要更理性一些，结合本周情况，但其实也没有要记很多，更多需要好好思考一下下周的安排计划。

月度总结

月度总结单独一页，相比日总结和周总结的简单来说，月度总结就要极度理性来分析自己这个月的情况。

我个人是安排在每个月的1号总结，并计划下个月的事项。

橙本的月度总结主要通过生活、学习、项目、财务、健康、关系六个方面进行统计和思考：

  • 学习（一般情况下不设deadline）：
    • 学习到的东西
    • 学习时的状态
    • 是否需要有调整的点
    • …
  • 项目（如博客搭建）：
    • 项目完成度汇总
    • 是否需要有调整的点
    • …
  • 财务：
    • 财务现状
    • 本月收支情况
    • 理财情况
    • 副业情况
    • …
  • 健康：
    • 完成度如何
    • 是否有需要调整的点
  • 关系：
    • 多和家人朋友交流
    • 是否有没回信息的情况
    • …
  • 生活
    • 当前生活状态
    • 有没有需要保持或改进的点
  • 整体：汇总总结一下

下个月的月度计划以16号为节点，分为上半月和下半月进行安排制定：

  • 结合月度总结情况，在完成一部分即可直接在下一段写上想法，计划措施；
  • 学习和项目方面只做优化改进，调整状态，不做Deadline；
  • 财务方面按照季度目标严格执行；
  • 每12月1日、3月1日、6月1日和9月1日结合六个方面制定相关的季度方向和目标；
  • 季度目标写在每月总结的开头，每个月都首先写好本季度方向；

不断优化，坚定执行

不断优化，坚定执行，收获的将不只是这些具体的点，虽然知识的不断长进，习惯慢慢养成这些很重要，但其实最大的收获将是背后的思维转变，主动的思考可能会有点累，但当思考成了对待世界的态度时，便不会觉得很累。

只要目标制定了，事项安排了。除非特殊情况，无论对错本次都按照此目标和计划实行，如果觉得不合适需要修改可以在下一次制定的时候考虑到。人的一生很长，对于这种问题有很多时间来修改完善，所以更不能把此作为拖延逃避的借口。

一开始可能很多都不完善，但其实也没有必要放弃，不断优化到它变成我们目前可以接受的程度，自然而然就接受了，也自然而然就变得更好了，可能这才是顺其自然的最终解。

加油咯💪。
